sphinxcontrib-sysml
===================

Sphinx extension for SysML v2 need types and diagrams via sphinx-needs and PlantUML.

Installation
------------

.. code-block:: bash

   pip install sphinxcontrib-sysml

Usage
-----

Add to your ``conf.py``:

.. code-block:: python

   extensions = [
       "sphinx_needs",
       "sphinxcontrib.sysml",
   ]
